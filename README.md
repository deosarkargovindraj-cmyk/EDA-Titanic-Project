# EDA + Logistic Regression on Titanic Dataset

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset and builds a Logistic Regression model to predict survival.

## Tasks Performed
- Data Cleaning
- Univariate Analysis
- Bivariate Analysis
- Data Visualization
- Logistic Regression Model

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results
- Females had higher survival rate
- Higher class passengers survived more
- Model achieved good accuracy
